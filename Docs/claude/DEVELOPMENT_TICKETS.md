# シフト管理アプリ 開発チケット一覧

## 概要

このドキュメントは、シフト管理アプリの開発を管理可能な単位に分割したチケット一覧です。
各チケットには、実装の詳細、依存関係、受け入れ基準が含まれています。

---

## フェーズ1: 基盤構築

### TICKET-001: プロジェクト初期化

**優先度**: 最高
**依存関係**: なし

**タスク**:
- Next.js 14+ プロジェクトの作成（App Router）
- TypeScript 設定
- Tailwind CSS 設定
- ESLint / Prettier 設定
- ディレクトリ構造の作成

**受け入れ基準**:
- [ ] `npm run dev` でアプリが起動する
- [ ] TypeScript のコンパイルエラーがない
- [ ] ESLint / Prettier が正常に動作する
